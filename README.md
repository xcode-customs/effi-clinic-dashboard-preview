# Effi Clinic Dashboard Preview

Предварительный клиентский дашборд по Этапу 1.

В репозитории опубликован только клиент-безопасный HTML-отчет:

- без исходных Telegram-переписок;
- без локальных путей к рабочим файлам;
- без выгрузок, таблиц и приватных материалов;
- с первичными срезами, которые не являются финальным аудитом.

Финальные выводы, причины потерь и рекомендации по скриптам должны быть добавлены после следующего этапа анализа.
